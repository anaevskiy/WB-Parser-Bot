# Бот-парсер товаров WB

Данный бот парсит товары для пользователя исходя из его
запроса и геолокации. Этот проект не несет за собой какой-то
реальной пользы для анализа, он был создан для 
демонстрации моих возможностей в сфере разработки ботов.
Иначе говоря для портфолио


### Функционал

* Сбор пользовательских данных (геолокация, наименование товара)
* Отправка запросов на сервера Wildberries для получения данных
* Формирования excel-таблицы со всеми полученными данными
* Введение журнала (всех действий всех пользователей)
* Авто отправка личного сообщения админу при возникновении ошибки

## Инструкция по установке

Если у вас появилось желание запустить данного бота у себя на устройстве,
тогда вам нужно будет выполнить несколько шагов


<br>

#### Установка Python 3.11

Поскольку бот написан на Python, вам необходимо поставить
его на компьютер. Для этого заходим на сайт
[Python](https://www.python.org/) и скачиваем нужную нам версию. 
В установщике поставьте галочку на "ADD TO PATH"

<br>

#### Получение токена бота

В Telegram заходим в [@BotFather](https://t.me/BotFather)
прописываем /newbot и следуем инструкции. В итоге вы получите
от бота так называемый API TOKEN. Скопируйте его и вставьте в
файл .env после "BOT_TOKEN="

<br>

#### Получение токена сервиса DADATA

Данный сервис позволяет получать одни геоданные на основе других
данных связанных с местоположением. Поэтому нам необходимо 
получить API TOKEN для того чтобы бот мог корректно работать.

Заходим на [сайт](https://dadata.ru/api/geolocate/), 
нажимаем кнопку "Войти" и регистрируемся, далее на первоначальной
странице переходим на вкладку "Python", находим строку "token="
и копируем то что в кавычках

Далее переходим .env и после "DADATA_TOKEN=" вставляем наш токен

<br>

#### Получение ID Telegram

ID нужен для того чтобы назначить человека или несколько людей админом в боте.
Это лишь влияет на получение сообщений об ошибках от бота. Если
вам это не надо, пропускайте этот пункт

Ну, а если все таки хотите назначить себя или еще пару человек
админом, тогда перейдите в специального [бота](https://t.me/getmyid_bot), 
напишите ему что-нибудь и в ответе число после "Your user ID:" - ваш ID. 
Для получения ID другого человека, просто перешлите этому боту
сообщение который писал тот человек и в ответе число после
"Forwarded from:" - ID того человека.

Далее зайдите в .env и после "ADMINS=" укажите все ID админов через запятую. Ну или же просто ID, если будет один админ

<br>

#### Последний рывок

Нам осталось выполнить пару команд в командной строке:

1. cd <путь папки с ботом>
2. pip install -r requirements.txt
3. python run.py


<br>

Вот и все! Бот запущен и вы можете его протестировать :)
